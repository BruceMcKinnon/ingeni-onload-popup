Display a Foundation Reveal Popup when visit a site, or when you login



== Description ==

Allows you to display a Foundation Reveal popup when the user visits your site.

* - Specify how many days between showing the popup

* - Specify the end date of the popup

* - Specify whether the popup will be show to everyone, or only logged in users



== Requirements ==

* - Requires a theme based on Zurb Foundation which supports the Reveal popup component.



== Installation ==

1. Upload the 'ingeni-onload-popup' folder to the '/wp-content/plugins/' directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Change the message, end date and other settings via the Tools > Ingeni Popup form.



== Frequently Asked Questions ==

= Are there any requirements? =

Your Wordpress  theme must support both jQuery and Foundation, including the Foundation Reveal popup component.
